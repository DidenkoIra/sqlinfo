# Web SQL INFO
Это групповой проект, выполенный в рамках обучения в школе 21.
Для реализации web-интерфейса использован SpringMVC.

## Build with Maven
```
cd SQLInfo
mvn clean package
```

## Run with docker-compose
```
cd ..
docker-compose up -d --build
```


## Visit page 
http://localhost/sqlinfo
